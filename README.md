## 🚀 Installation
```
npm i multilingual-date-picker
```

## ⚙️ Usage

Import package
import multilingual-date-picker from 'multilingual-date-picker';
```javascript
export default {
  components: {
    MultilingualDatepicker
  }
}
```


Create component in your template
```javascript
<MultilingualDatepicker />
```
